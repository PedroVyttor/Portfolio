Sobre Mim

Meu nome é Pedro Vyttor Pessoa Alves, tenho 19 anos e curso Ciência da Computação. Tenho grande interesse na área de jogos e, antes mesmo de ingressar no curso, já explorava esse mundo desenvolvendo homebrews para o console PlayStation 2.

Entre esses projetos, criei um aplicativo simples de teste de controles (DualShock), voltado para verificar o funcionamento dos botões e analógicos, embora atualmente eu não consiga testá-lo por não possuir um PS2 em funcionamento.

Além disso, também desenvolvo temas personalizados para PlayStation 4 no formato PKG. Essas experiências, junto com outros projetos e curiosidades na área, despertaram meu interesse pela computação e pelo desenvolvimento de software.

Projetos

Projeto 1

Este site simples desenvolvido com HTML e CSS (Portfólio).

Projeto 2 - Homebrew para PlayStation 2

Desenvolvimento de um aplicativo (homebrew) para o console PlayStation 2, focado em testar controles DualShock. O objetivo era verificar o funcionamento dos botões e analógicos, auxiliando na identificação de possíveis defeitos no controle. O projeto foi desenvolvido antes da minha entrada no curso de Ciência da Computação.

Projeto 3 - Temas personalizados para PlayStation 4

Criação de temas personalizados para PlayStation 4 no formato PKG. Esse projeto envolve edição de imagens, organização de arquivos do sistema e personalização da interface do console, permitindo alterar aparência, ícones, música e plano de fundo.
